HR Retention / Attrition Streamlit App
======================================

Files included:
- app.py : main Streamlit application (run with `streamlit run app.py`)
- requirements.txt : packages required (no pinned versions)
- HR_sample.csv : small sample dataset to try the app immediately (you can replace with your HR data)
- README.md : this file

Instructions:
1. Extract files to repo root (no folders).
2. Push to GitHub and connect the repository to Streamlit Cloud, or run locally with: `streamlit run app.py`
3. In the sidebar, map your columns: Job Role, Satisfaction (numeric), and Target (Attrition).
4. The app drops nulls automatically. For different handling, modify app.py accordingly.
